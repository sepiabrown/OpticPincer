# Social preview text

OpticPincer gives LLM coding agents (AI coding assistants) the ability to see and interact with native desktop GUI applications. Instead of guessing pixel coordinates, it reads an exported UI widget tree and clicks elements by name. Built for developers who need automated GUI design verification and debugging without leaving their AI workflow.
